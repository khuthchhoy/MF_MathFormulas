Functions: $f,g,y$
Position of an object: $s$
Velocity: $v$
Acceleration: $w$
Independent variable: $x$
Time: $t$
Natural number: $n$

Velocity and Acceleration

$s=f(t)$ is the position of an object relative to a fixed coordinate system at a time $t$,
$v=s'=f'(t)$ is the instantaneous velocity of the object,
$w=v'=s''=f''(t)$ is the instantaneous acceleration of the object.

Tangent Line

$$y-y_0=f'\Big( x_0\Big)\Big( x-x_0\Big)$

Normal Line

$$y-y_0=-\dfrac{1}{f'\Big(x_0\Big)}\Big( x-x_0\Big)$$

Increasing and Decreasing Functions

If $f'\left( x_0\right)\gt 0$, then $f(x)$ is increasing at $x_0$
If $f'\left( x_0\right)\lt 0$, then $f(x)$ is decreasing at $x_0$
If $f'\left( x_0\right)$ does not exist or is zero, the the test fails.

Local extrema

A function $f(x)$ has a local maximum at $x_1$ if and only if there exists some interval containing $x_1$ such that $f\Big( x_1\Big)\ge f(x)$ for all $x$ in the interval.

A function $f(x)$ has a local minimum at $x_2$ if and only if there exists some interval containing $x_2$ such that $f\Big( x_2\Big)\ge f(x)$ for all $x$ in the interval.

Critical Points

A critical point of $f(x)$ occurs at $x_0$ if and only if either $f'\left( x_0\right)$ is zero or the derivative doesn't exist.

First Derivative Test for Local Extrema.

If $f(x)$ is increasing $\Big( f'(x)\gt 0\Big)$ for all $x$ in some interval $\left( a, x_1\right]$ and $f(x)$ is decreasing $\Big( f'(x)\lt 0\Big)$ for all $x$ in some interval $\left[ x_1,b)$, then $f(x)$ has a local maximum at $x_1$.

If $f(x)$ is decreasing $\Big( f'(x)\lt 0\Big)$ for all $x$ in some interval $\left( a, x_2\right]$ and $f(x)$ is increasing $\Big( f'(x)\gt 0\Big)$ for all $x$ in some interval $\left[ x_2,b)$, then $f(x)$ has a local minimum at $x_2$.

Second Derivative Test for Local Extrema.

If $f'\left( x_1\right)=0$ and $f''\left( x_1\right)\lt 0$, then $f(x)$ has a local maximum at $x_1$.
If $f'\left( x_2\right)=0$ and $f''\left( x_2\right)\gt 0$, then $f(x)$ has a local minimum at $x_1$.

Concavity

$f(x)$ is concave upward at $x_0$ if and only if $f'(x)$ is increasing at $x_0$.

$f(x)$ is concave downward at $x_0$ if and only if $f'(x)$ is decreasing at $x_0$.

Second Derivative Test for Concavity

If $f''(x_0) > 0$, then $f(x)$ is concave upward at $x_0$.

If $f''(x_0) < 0$, then $f(x)$ is concave downward at $x_0$.

If $f''(x_0) = 0$ or $f''(x_0)$ does not exist, then the test is inconclusive.

Inflection Points

If $f'\left(x_3\right)$ exits and $f''(x)$ changes sign at $x=x_3$, then the point $\Big( x_3,f(x_3) \Big)$ is an inflection point of the graph of $f(x)$.

If $f''\left(x_3\right)$ exists at the inflection point, then $f''\left( x_3\right)=0$

L'Hôpital's Rule

Suppose $\lim_{x \to a} f(x) = 0$ and $\lim_{x \to a} g(x) = 0$, or $\lim_{x \to a} f(x) = \pm\infty$ and $\lim_{x \to a} g(x) = \pm\infty$.

If $f$ and $g$ are differentiable near $a$ and $g'(x) \ne 0$, then $$\displaystyle \lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)}$$

provided the limit on the right side exists or is $\pm\infty$.
