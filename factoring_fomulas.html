Factoring Formulas

$a,b,c$ are real number and $n$ is natural number.

$$\begin{aligned} &a^2 - b^2 = (a-b)(a+b) \\\\ &a^3 - b^3 = (a-b)\left( a^2 + ab + b^2\right) \\\\ &a^3 + b^3 = (a+b)\left( a^2 - ab + b^2 \right) \\\\ &a^4 - b^4 = \left( a^2 - b^2 \right) \left( a^2 + b^2 \right) = (a - b)(a + b)\left( a^2 + b^2 \right)\\\\ &a^5 - b^5 = (a - b)\left( a^4 + a^3b + a^2b^2 + ab^3 + b^4\right) \\\\ &a^5 + b^5 = (a + b)\left( a^4 - a^3b + a^2b^2 - ab^3 + b^4\right) \\\\ &\text{If $n$ is odd, then} \\\\ &a^n + b^n = (a + b)\left(a^{n-1} - a^{n-2}b + a^{n-3}b^2 -\cdots - ab^{n-2} + b^{n-1} \right) \\\\ &\text{If $n$ is even, then} \\\\ &a^n - b^n = (a - b)\left(a^{n-1} + a^{n-2}b + a^{n-3}b^2 + \cdots + ab^{n-2} + b^{n-1} \right) \\\\ &a^n + b^n = (a + b)\left(a^{n-1} - a^{n-2}b + a^{n-3}b^2 -\cdots + ab^{n-2} - b^{n-1} \right) \end{aligned}$$
