Functions: $f,y,u,v$
Independent variable: $x$
Natural number: $n$

  • Second derivative $$f'' = \Big( f'\Big)' = \Big(\dfrac{\mathrm{d} y}{\mathrm{d} x}\Big)' = \dfrac{\mathrm{d}}{\mathrm{d}x}\Big(\dfrac{\mathrm{d}y}{\mathrm{d}x} \Big) = \dfrac{\mathrm{d}^2 y}{\mathrm{d}x^2}$$
  • Higher-Order derivative $$f^{(n)}=\dfrac{\mathrm{d}^ny}{\mathrm{d}x^n}=y^{(n)}=\Big(f^{(n-1)} \Big)'$$
  • $(u+v)^{(n)} = u^{(n)} + v^{(n)}$
  • $(u-v)^{(n)} = u^{(n)} - v^{(n)}$
  • Leibnitz's Formulas $$\begin{aligned} &(uv)'' = u''v + 2u'v' +uv'' \\\\ &(uv)''' = u'''v + 3u''v' + 3u'v'' + uv''' \\\\ &(uv)^{(n)} = u^{(n)} + nu^{(n-1)}v' + \\ &\phantom{(uv)^{(n)} =}\frac{n(n-1)}{1\cdot 2}u^{(n-2)}v'' + \cdots + uv^{(n)} \end{aligned}$$
  • $\Big(x^m \Big)^{(n)} = \dfrac{m!}{(m-n)!}x^{m-n}
  • $\Big( x^n\Big)^{(n)} = n!$
  • $\Big( \log_a x\Big)^{(n)} = \dfrac{(-1)^{n-1}(n-1)!}{x^n\ln a}$
  • $\Big( \ln x\Big)^{(n)} = \dfrac{(-1)^{n-1}(n-1)!}{x^n}$
  • $\Big(a^x \Big)^{(n)} = a^x\ln^n a$
  • $\Big( e^x\Big)^{(n)} = e^x$
  • $\Big(a^{mx} \Big)^{(n)} = m^na^{mx}\ln^n a$
  • $\Big( \sin x\Big)^{(n)} =\sin\Big(x+\frac{n\pi}{2} \Big) $
  • $\Big( \cos x\Big)^{(n)} =\cos\Big(x+\frac{n\pi}{2} \Big) $
