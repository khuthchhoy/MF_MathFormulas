Functions: $f,g,u,v$
Independent variables: $x,t,\xi$
Indefinite integral of a function: $\displaystyle\int f(x)dx, \int g(x)dx,...$
Derivative of a function: $y'(x),f'(x), F'(x),...$
Real constants: $C,a,b,c,d,k$
Natural numbers: $m,n,i,j$

Indefinite Integral

  1. $\displaystyle\int f(x)dx = F(x) + C$ if $F'(x) = f(x)$
  2. $\Big( \displaystyle\int f(x)dx \Big)'dx = f(x)$
  3. $\displaystyle\int kf(x)dx = k\int f(x)dx$
  4. $\displaystyle\int\Big[f(x) + g(x) \Big] dx = \int f(x)dx + \int g(x)dx$
  5. $\displaystyle\int\Big[ f(x) - g(x)\Big] dx = \int f(x)dx - \int g(x) dx$
  6. $\displaystyle\int f(ax)dx = \dfrac{1}{a}F(ax) + C$
  7. $\displaystyle\int f(ax+b) dx = \dfrac{1}{a}F(ax+b) + C$
  8. $\displaystyle\int f(x)f'(x) dx = \dfrac{1}{2}f^2(x) + C$
  9. $\displaystyle\int \dfrac{f'(x)}{f(x)} dx = \ln\Big| f(x)\Big| + C$
  10. Method of Substitution
    If $x=u(t)$ then, $$\displaystyle\int f(x) dx = \int f\Big(u(t) \Big)u'(t)dt$$
  11. Integration by Parts $$\displaystyle\int udv = uv - \int vdu,$$ where $u(x),v(x)$ are differentiable functions.

Integrals of Rational Functions

  1. $\displaystyle\int adx = ax + C$
  2. $\displaystyle\int xdx = \dfrac{x^2}{2} + C$
  3. $\displaystyle\int x^2dx = \dfrac{x^3}{3} + C$
  4. $\displaystyle\int x^p dx = \dfrac{x^{p+1}}{p+1} + C,\,p\ne -1$
  5. $\displaystyle\int \Big( ax+b\Big)^ndx = \dfrac{(ax+b)^{n+1}}{a(n+1)} + C,\,n\ne -1$
  6. $\displaystyle\int \dfrac{dx}{x} = \ln|x| + C$
  7. $\displaystyle\int \dfrac{dx}{ax+b} = \dfrac{1}{a}\ln|ax+b| + C$
