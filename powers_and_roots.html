Powers

Bases (positive real numbers): $a,b$
Powers (rational numbers): $m,n$

$$\begin{aligned} &a^ma^n = a^{m+n} \\\\ &\frac{a^m}{a^n}=a^{m-n} \\\\ &(ab)^n = a^nb^n \\\\ &\left( \frac{a}{b}\right)^n = \frac{a^n}{b^n} \\\\ &\left( a^m\right)^n = a^{mn} \\\\ &a^0 =1\,\quad a\ne 0 \\\\ &a^{-n} = \frac{1}{a^n} \\\\ &\left( \frac{a}{b}\right)^{-n} = \left( \frac{b}{a}\right)^n = \frac{b^n}{a^n} \\\\ &a^{\frac{m}{n}} = \sqrt[n]{a^m} \\\\ \end{aligned}$$

Roots

Bases: $a,b$
Powers (rational numbers) $m,n$
$a,b \ge 0$ for even roots $(n=2k,\, k\in \mathbb{N})$

$$\begin{aligned} &\sqrt[n]{ab} = \sqrt[n]{a}\sqrt[n]{b} \\\\ &\sqrt[n]{a}\sqrt[m]{b} = \sqrt[mn]\sqrt{a^mb^n} \\\\ &\sqrt[n]{\frac{a}{b}} = \frac{\sqrt[n]{a}}{\sqrt[n]{b}},\quad b\ne 0\\\\ &\frac{ \sqrt[n]{a} }{ \sqrt[m]{b} } = \frac{ \sqrt[mn]{a^m} }{ \sqrt[mn]{b^n} } = \sqrt[mn]{ \frac{a^m}{b^n} },\quad b\ne 0 \\\\ &\left( \sqrt[n]{a^m} \right)^p = \sqrt[n]{a^{mp}} \\\\ &\left( \sqrt[n]{a} \right)^n = a \\\\ &\sqrt[n]{a^m} = \sqrt[np]{a^{mp}} \\\\ &\sqrt[n]{a^m} = a^{\frac{m}{n}} \\\\ &\sqrt[m]{ \sqrt[n]{a} } = \sqrt[mn]{a} \\\\ &\left( \sqrt[n]{a} \right)^m = \sqrt[n]{a^m} \\\\ &\frac{1}{\sqrt[n]{a}} = \frac{\sqrt[n]{a^{n-1}} }{a},\quad a\ne 0\\\\ &\sqrt{a\pm \sqrt{b}} = \sqrt{\frac{a+\sqrt{a^2-b} }{2} }\pm \sqrt{\frac{a-\sqrt{a^2-b} }{2} } \\\\ &\frac{1}{ \sqrt{a}\pm \sqrt{b} } = \frac{ \sqrt{a} \pm \sqrt{b} }{a-b} \end{aligned}$$
